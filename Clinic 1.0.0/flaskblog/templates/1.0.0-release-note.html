1.0.0 发行说明


发布于 2020-03-25

有什么新功能？

  • 患者模块
    • 医生可以点击“添加患者”添加新患者信息
    • 医生还可以在患者信息里添加添加患者病患信息
    • 医生可以点击“查找患者”来查找全部患者
    • 管理员账户具有更改与删除患者个人与病患信息，此功能仅限于管理员账户，除此账户外，其他医生只拥有更改的权限
  • 药物
    • 医生可以添加药物并更改药物信息
    • 大夫还可以使用“查找药物”来查找所需的药物
    • 管理员账户有权利更改并删除全部药品信息。并非管理员并无此功能
  • 工作日志
    • 医生可以添加工作日志以记录工作内容
    • 医生还可以查看全部属于自己的工作日志
  • 个人账户
    • 医生可以更改密码
    • 医生还可以查看全部属于自己的工作日志
  • 管理员账户
    • 管理员可以更改删除患者/患者信息/药品/医生
    • 只有管理员可以添加新医生
  • 新功能
    • 医生可以使用电子邮箱重制密码
    • 后台使用患者生日自动计算患者年龄
    • 404 页面
    • 筛选出但一患者的历史记录

    • 患者病患信息错误
    • 患者病患信息添加完的重新导向错误
    • 查找患者中文搜索
    • 登陆转接错误
    • 医生更改密码值错误
    • 管理员更改密码值错误
    • 工作日志内容格式错误修复
    • 管理员账户具有更改与删除患者个人与病患信息，此功能仅限于管理员账户，除此账户外，其他医生只拥有更改的权限。
